# M2HRI: Multimodal Multi-Agent Human-Robot Interaction ### An LLM-Driven Framework for Personalized Multi-Robot Interaction **Shaid Hasan, Breenice Lee, Sujan Sarker, Tariq Iqbal** University of Virginia [📄 Paper](./M2HRI.pdf) | [🌐 Project Page](https://project-m2hri.github.io/) | [💻 Code (Coming Soon)] --- ## Overview Multi-robot systems are increasingly deployed in social environments, yet most existing approaches treat robots as identical agents. In this work, we introduce **M2HRI**, a multimodal multi-agent framework that enables: - Distinct **robot personalities** - **Long-term memory** for personalization - **Multimodal perception** (vision + speech) - **Centralized coordination** for coherent interaction Our results show that modeling **agent individuality + coordination** significantly improves interaction quality, engagement, and personalization. --- ## Demo Video **What this demo shows:** - Two robots with **distinct personalities** - Real-time **speech + vision perception** - **Turn-taking coordination** - **Embodied interaction** (speech, gestures) - Human interacting with both agents simultaneously
